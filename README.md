To install on Linux:  

    $ cd ~  
    $ git clone https://github.com/danhui/vimrc.git .vim  
    $ echo "runtime vimrc" > .vimrc  
    $ vim  


To install on Windows:

    $ cd %USERPROFILE%  
    $ git clone https://github.com/danhui/vimrc.git vimfiles  
    $ echo "runtime vimrc" > .vimrc  
    $ vim
